# Gushwork Frontend Assignment

A frontend assignment built with **React** and **Vite** for the Tagmango project.

This project is a **responsive landing page** built using **HTML, CSS, and Vanilla JavaScript**.  
The goal of this assignment was to implement a clean, responsive UI with interactive components and deploy it online.

## 🚀 Live Demo

👉 https://gushwork-fe-assignment.vercel.app

## 🧩 Features

- Fully responsive layout (Mobile, Tablet, Desktop)
- Clean and semantic HTML structure
- Modern CSS styling
- Interactive components using Vanilla JavaScript
- FAQ accordion functionality
- Smooth UI interactions and hover effects

## 📦 Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Vercel** (Deployment)
